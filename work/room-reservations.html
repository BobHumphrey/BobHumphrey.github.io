Room Reservations

Room Reservations lets students and faculty at The University of North Carolina Wilmington reserve group study rooms at Randall Library. The application was written as a Drupal module so that it would work seamlessly with the rest of the library website. The web app needed to be one that patrons could easily use without requiring assistance from library staff. A project like this is all about the business rules: permitting reservations, but only when the library is open, and only for the allowable length of time, and only for rooms that havenâ€™t been already reserved, and so on and so on.
