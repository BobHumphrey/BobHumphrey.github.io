Information Literacy

When Randall Library was creating an Information Literacy portal, using Drupal 7, my boss wanted a custom, responsive theme that would not be dependent on any other module, such as Zen or Omega. I decided to build the theme using Sass, Compass and Susy because these tools made it easy to work with the multitude of ID and class names already contained within the Drupal HTML code.
