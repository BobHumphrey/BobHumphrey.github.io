Baseball Statistics

A lot of my work at Randall Library involves writing applications for internal use within the library, keeping track of things like the daily foot traffic into the facility, usage of the 100+ public computers in the Learning Commons, and the number of classes being taught by the library faculty. My tool of choice for these kinds of projects is the Yii PHP Framework. Using this type of a framework offers a lot of benefits, the most important of which is faster development. With a good framework, a lot of the common, boring, everyday coding is already done so that you can concentrate on the unique challenges of the task at hand. Frameworks also help you organize your code in the same way for every job so that future updates and changes are a lot easier to make. I’ve tried a lot of different frameworks, and they all have their advantages and disadvantages, but Yii has been the best one I have found so far. It’s fairly difficult and time consuming to learn, but once you’ve gained some experience with it, it’s like you’ve developed super powers for building web applications.

I wrote the Baseball Statistics application in a couple of weekends so that I would have a nice example of the kind of work that can quickly be done using the Yii framework. If you’re interested, take a look at the code here. You can use the application to explore 142 years of baseball history. Every major league player, team, and franchise from 1871 to 2012 is included. Use the form provided to select a particular year, player, or franchise. Or just follow the links on every page to jump around and learn about the various players and teams.
