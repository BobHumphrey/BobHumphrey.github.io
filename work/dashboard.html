Dashboard

When Randall Library was interested in creating a Statistics Dashboard to “to visually demonstrate the use, activity and resources in the library,” I put together the following solution. Making use of Drupal blocks, I coded PHP scripts that called upon the Open Flash Chart 2 API to create attractive, easy to use charts and graphs. A few of the charts are manually updated as new data becomes available, but most of the graphs query live databases to display current information that is continuously updated. The final result is an attractive, easy to understand page made entirely of free, open source components.
